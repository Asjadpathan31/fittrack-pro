# FitTrack Pro 🏋️

A full-stack Health & Fitness web application built with the MERN stack.

## Features
- 🔐 JWT Authentication (Login/Signup)
- 💪 Workout Logger with Exercise Suggestions
- 🔥 Auto Calorie Calculator
- 📊 Progress Charts with Recharts
- 🏃 BMI Calculator
- 🔥 Workout Streak Tracker
- 🎨 Animated UI with Framer Motion

## Tech Stack
- **Frontend:** React.js, Framer Motion, Recharts, Lucide Icons
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Mongoose ODM)
- **Auth:** JWT (JSON Web Tokens)
- **Deployment:** Vercel + Render

## Setup

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
